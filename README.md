INTRODUCTION

Face IO represents a smart attendance system incorporating facial identification technology. This system is not only time-saving but also eliminates the potential for proxy attendance through facial authorization. The implementation utilizes OpenCV and a Convolutional Neural Network (CNN) trained model to fulfill the requirements of this intelligent attendance system

SYSTEM FEATURES

1. Utilizes advanced facial recognition for accurate attendance tracking.
2. Streamlines attendance recording for increased efficiency.
3. Enhances security by eliminating proxy attendance through facial authorization.
4. Incorporates OpenCV for improved facial recognition capabilities.
5. Provides instant updates on attendance records for timely monitoring.

TECNICALITIES/TOOLS USED

1. Deep Learning
2. Machine Learning
3. OpenCV
4. SQL Database
5. Python
